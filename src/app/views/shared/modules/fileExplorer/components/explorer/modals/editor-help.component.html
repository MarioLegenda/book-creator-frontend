OVERVIEW

Welcome to the Rebel Source code editor. In any regard, this code editor is the same as any desktop code editor (IDE) that you are using although, it does not have all the advanced features that they have. But the basics are there and I hope you find it useful.

DIRECTORIES AND FILES

Creating, removing, copying and deleting directories and files are the basic operations of a code editor and this editor is not different than any other. How this code editor works, you can see it this video.

The help video will also explain how to run the code. Mind you that the default timeout when running code is 10 seconds for interpreted languages like Javascript or Ruby and 15 seconds for compiled languages like Go or Rust.

Hope you find it useful.
