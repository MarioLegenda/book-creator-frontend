Main header block help and how to's

A code block allows you to create the code that your readers can execute. Every code block must have either an environment (Javascript, Go or some other) or an imported code project. An environment allows readers to execute a single piece of runnable code while a code project is a project that consist of many files that the user can then import. If you which to import a code project, you have to create it first in the code projects dashboard. Lets know explore more how can we create a new environment.
